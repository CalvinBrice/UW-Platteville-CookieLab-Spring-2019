To control cookie robot

wristRotationCCW	CCW rotation of gripper wrist	1	BUTTON	Button 3	0	0.0	0.0
x_rotation	Stores servo angle of gripper base rotation	3	SLIDER	X Rotation	0	1.0	0.0
verticalTravel	Stores Nema 23 vertical height position	3	SLIDER	Y Axis	0	1.0	0.0
